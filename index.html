~/dotfiles

A reproducible developer environment built on Zsh · mise · just · uv, wired for multi-agent AI workflows and a complete local-dev stack — datastore & SaaS emulators, OpenTelemetry observability, and stable https://*.localhost routing. One command on a fresh machine.

maintained by hironow · updated 2026-05-30 · MIT

▾ quick install source ↗
$ bash -c "$(curl -fsSL https://raw.githubusercontent.com/hironow/dotfiles/main/install.sh)"

already cloned? run just install — or just deploy to relink on the current host.

§02

Local-dev stack

First-party, vendored, Docker-Composed. Bring up datastore & SaaS emulators with telemetry, and reach every HTTP UI by name over HTTPS — no port numbers to memorize.

emulator 13 services — Firebase, Bigtable, Spanner, pgAdapter, Postgres, Neo4j, Qdrant, Elasticsearch, MLflow, A2A & MCP inspectors. just emu-start
telemetry OpenTelemetry collector → Grafana · Prometheus · Tempo · Loki, on a shared OTel network. just tel-up
emulate 9 API emulators — GitHub, Google, Slack, Apple, Microsoft, Stripe, Clerk, Okta, Resend (ports 4100–4108). just emu-api
portless stable https://<name>.localhost routes via a trusted local CA — grafana.localhost, firebase.localhost, stripe.localhost … just portless-up · just portless-doc

§04

Commands

Everything routes through just — just alone (or just help) lists all task groups.

just installbootstrap the full environment on a new machine
just deploysymlink dotfiles onto the current host (idempotent)
just cifast gate — lint / format / semgrep rule self-tests / IaC tests (no Docker)
just testdev-container sandbox suite (snapshots tracked files only)
just emu-start · emu-apibring up datastore emulators · API/SaaS emulators
just tel-up · portless-uplaunch telemetry · register .localhost routes
just emu-stop · tel-down · emu-api-stoptear each stack back down
